## ⭐ TUKTAK (In Progress)

> **AI-powered Interior Repair Platform**

AI를 활용한 인테리어 견적 생성부터 시공자 매칭까지 지원하는 서비스입니다.

### 주요 기능

- 🤖 AI 견적 생성
- 📑 AI 리스크 리포트
- 🔥 실시간 시공자 매칭
- 🏗 시공 진행 관리
- ⭐ 리뷰 및 평점 시스템

### Architecture

```text
Damage Image
      │
      ▼
AI Estimate
      │
      ▼
Risk Report (LLM + RAG)
      │
      ▼
Real-time Matching
      │
      ▼
Work Order
      │
      ▼
Review
```
